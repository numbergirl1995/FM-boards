# docs — GIM Boards 設計文書の目次

このフォルダは GIM Boards の設計文書を集約している。**作問・実装のたびに「どれが最新か」で迷わないための目次。**
（GitHub ではフォルダを開くとこの README が先頭に表示される。）

---

## ✅ 最新版（これを使う）

作問エージェント（Cowork 等）や Claude Code に渡すのは、以下の最新版のみ。

| ファイル | 役割 | いつ使う |
|---|---|---|
| **category_taxonomy_v2.md** | カテゴリー分類の背骨（**24カテゴリー**・imCore・マイナー科の範囲限定） | 分類を参照するとき／作問時 |
| **authoring_rules_v3.md** | 作問グラウンドルール（作り方の指針。PICOタブ・解説の太字・図表・画像版権・出力形式・24カテゴリー基準） | 作問時 |
| **問題データ_統一スキーマ仕様_雛形_v2.md** | 問題データのスキーマ・雛形（単一 `category`／`explain`／`evidence[].pico` 等） | 作問時（1問の形を作る） |

## 🗄 旧版（参照用・作問には使わない）

過去の経緯を追うために残している。**作問エージェントには渡さない**（最新版と混ざると混乱するため）。

| ファイル | 置き換え先 |
|---|---|
| カテゴリー分類_確定仕様_v1.md（20カテゴリー） | → category_taxonomy_v2.md |

※初代の作問グラウンドルール・スキーマ v1 は、リポジトリには未追跡（docs 未収録）のまま。最新は上記の v3／v2。

## メモ
- 新しいバージョンを作ったら、この README の「最新版」表を更新し、旧版を「旧版」表へ移す。
- 分類・ルール・スキーマは相互に依存する（分類 → スキーマの category 値 → 作問ルールの領域基準）。バージョンを上げるときは整合を確認する。
